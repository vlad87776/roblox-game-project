## Core loop
1. Игрок заходит в матч/сервер.
2. Выполняет основную активность (например: проходит этапы, фармит ресурсы, строит базу).
3. Получает награду (валюта/опыт/предметы).
4. Улучшает персонажа/базу/инвентарь.
5. Возвращается в loop с более сложной целью.

## Основные механики
- Movement и взаимодействие с объектами.
- Прогрессия игрока (уровни/апгрейды).
- Экономика (soft currency).
- Система наград (daily/quest/achievement).

## Уникальность проекта
- Ставка на понятный onboarding и быстрый старт.
- Короткие игровые сессии (2–5 минут).
- Регулярные контент-апдейты.
