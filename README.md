# roblox-game-project

Репозиторий разработки Roblox-игры в Roblox Studio.

## О проекте
Этот репозиторий используется как devlog и документация по игре:
- концепт и цели проекта;
- план фич и обновлений;
- ссылки на Roblox place/experience;
- журнал изменений по версиям.

## Текущий статус
**Stage:** Prototype (в активной разработке)

## Что здесь будет
- `docs/game-design.md` — геймдизайн-док (core loop, механики, прогрессия).
- `docs/roadmap.md` — дорожная карта по версиям.
- `docs/place-ids.md` — IDs и ссылки на place/universe.
- `CHANGELOG.md` — заметки по обновлениям.
